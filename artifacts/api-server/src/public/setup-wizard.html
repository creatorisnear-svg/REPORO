Grant Bot Access to Your Server Host

Before connecting, you need to allow Aviv Bot to access your Rust server's RCON.

GPortal Go to your GPortal dashboard → Permissions → add avivbot as a user with RCON access.
Nitrado Go to your Nitrado dashboard → Guest Access Rights → add avivbot.
Other Hosts Enable WebRCON on your Rust server. You will need your RCON IP, port (usually 28016), and RCON password. Check your host's control panel for these details.
