Aviv Bot Documentation

Quick Start

Add Aviv Bot to your server, then run /setup to create all required roles and channels automatically.

Next, use /add-server with your RCON host, port, and password to connect your Rust server.

Assign the avivadmin role to server administrators and avivmod to moderators.

Setup Commands

/setup - Creates avivadmin, avivmod, avivlinked, vip-1, recycler-1, zorp-1 roles and all feed channels.

/add-server [server] [host] [port] [password] - Connect a Rust server via WebRCON.

/remove-server [server] - Deactivate a server connection.

/diag - Check RCON connection health and ping.

/aviv - Open the control panel embed.

/set [config] [value] - Change any bot setting (use autocomplete).

/configs - View all current settings for a server.

/admin-channels [feed] [channel] - Assign a Discord channel to a feed type.

/admin-positions list/add/remove - Manage teleport positions.

/admin-scheduler list/add/remove - Manage scheduled in-game messages.

Player Linking

Players must link their Discord to their in-game name to use economy, gambling, and other features.

/link [ingame_name] - Link your Discord account.

/unlink - Remove your link.

/admin-link [ingame_name] [discord_user] - Force-link a player (admin).

/whois - Look up a link by Discord user or in-game name.

/sync-me - Re-sync your list roles.

/get-playerinfo [ingame_name] - View all info about a player (admin).

Economy

/balance - Check your coin balance.

/daily - Claim a daily coin reward (20h cooldown).

/transfer [player] [amount] - Send coins to another linked player.

/swap [amount] [from] [to] - Move coins between servers.

/leaderboard - View the top 10 earners.

/setdailyscale [min] [max] - Set the daily reward range (admin).

/killpoints [type] [amount] - Set coins per kill (admin).

/add-points-player / /sub-points-player - Adjust one player's balance (admin).

/add-points-server / /sub-points-server - Adjust all players' balances (admin).

Gambling

/spin [bet] - Spin the bandit wheel. Outcomes: 3x (5%), 2x (15%), refund (10%), loss (70%).

/coinflip [bet] [heads/tails] - 50/50 coinflip.

/blackjack [bet] - Play blackjack against the dealer.

/maxbet [amount] - Set maximum bet limit (admin).

Shop

/shop - Browse the shop.

/admin-shop-create-shop - Initialize the shop.

/admin-shop-add-category [name] [type] - Add a category.

/admin-shop-add-subcategory [name] [parent] - Add a subcategory.

/admin-shop-add-item [name] [shortname] [price] [category] - Add an item.

/admin-shop-add-kit [kit_name] [price] [category] - Add a kit to the shop.

/delayshop [minutes] - Temporarily close the shop (wipe delay).

/openshop - Reopen the shop early.

Kits

/givekit [ingame_name] [kit_name] - Give a kit to a player via RCON (admin).

Kits are triggered automatically from in-game chat by the RCON parser based on config settings.

Lists

/add-to-list [list] [ingame_name] - Add a player to a list (mod).

/remove-from-list [list] [ingame_name] - Remove from a list (mod).

/get-list [list] - View all entries in a list (mod).

Common lists: viplist, zorpallowlist, zorpbanlist, prisonlist, recyclerlist, noteblocklist, elitelist1-elitelist44.

Prison

/prison [ingame_name] [duration_minutes] [reason] - Teleport to prison (mod).

/unprison [ingame_name] - Early release (mod).

/prison-list - View active prisoners (mod).

Configure prison TP position with /admin-positions add type:prison x: y: z:.

ZORP

/wipe-zorp - Delete all ZORP zones (admin).

/del-zorp [ingame_name] - Delete one player's zone (admin).

ZORP zones are tracked automatically when detected in RCON logs.

Raid Alerts

/raidlink [ingame_name] [frequency] - Register your base broadcaster frequency.

/list-raidlink - View all registered frequencies (mod).

/wipe-raidlink - Clear all registrations (admin).

/del-raidlink [ingame_name] - Remove one registration (admin).

When a frequency fires, the registered Discord user gets a DM raid alert automatically.

Moderation

/kick, /ban, /unban - Player management via RCON.

/mute, /unmute - In-game chat muting.

/warn [ingame_name] [reason] - Issue a warning.

/warnings [ingame_name] - View warning history.

/clearwarnings [ingame_name] - Clear all warnings (admin).
