Features

Everything your server needs

Built specifically for Rust Console Edition. Every feature connects directly to your game server through WebRCON.

🎁

Kit Builder

Create custom kits with items, quantities, and cooldowns. Deliver them to players via RCON directly from Discord with a single command.

💰

Economy System

Full coin economy with daily rewards, leaderboards, transfers, kill points, and cross-server coin swaps.

🔌

WebRCON Bridge

Real-time WebSocket RCON connection with auto-reconnect, idle drop, and persistent session management.

🚨

Raid Alerts

Automatic DM notifications when a player's base broadcaster fires. Per-player opt-in frequency registration.

🎲

Gambling

Bandit wheel, coinflip, and blackjack with configurable max bet, fair odds, and admin controls.

🗡

ZORP Integration

Full ZORP zone management with allow/ban lists, wipe commands, and automatic expiry tracking.

🔒

Prison System

Teleport players to a prison location with timed releases, DM notifications, and mod logs.

📺

Kill Feed

Parsed real-time kill feed posted to Discord with custom colors, phrases, and scientist kill tracking.

🛒

Shop System

Multi-category Discord shop with stock control, timers, kit bundles, and wipe delay support.

💬

Chat Bridge

Bi-directional chat relay between Discord and your Rust server. Players see Discord messages in-game.

🛡

Moderation

Kick, ban, mute, and warn players directly from Discord. Full audit logs and warning history.

📍

Teleports

Admin-configurable teleport positions with cooldowns. Directional TPs, prison, and custom locations.
